(usage)

Usage: lein javaapp

This AWS Simpleworkflow example requires that the example worker be running before the workflow is started. The simplest way to do this is open two terminal windows in this project directory, and execute each of the following (each in its own terminal):

$ lein javaapp start worker
$ lein javaapp run

Command execution order is not actually important here. If run is done before start worker, it will just be put in the execution queue on AWS until start worker is run (or until the run request times out at AWS).

Note that the worker will run continuously until its connection with AWS times out, whereas after the javaapp workflow is run, it will exit immediately.
